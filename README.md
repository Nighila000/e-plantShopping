# e-plantShopping

## Project Overview
e-plantShopping is a simple React-based e-commerce application for buying plants online. It allows users to browse different plant categories, view product details, and manage items in a shopping cart.

## Features
- Browse a list of plants
- Add items to cart
- View cart items
- Simple and responsive user interface

## Tech Stack
- React
- Redux
- JavaScript
- CSS

This project repository is named **e-plantShopping**.
